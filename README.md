# Employee Skill Searcher

Developed as a hackday project at [Clarifai](https://www.clarifai.com), this is a simple search engine to discover what skills your colleagues have and which people are willing to mentor others in those skills.

A live demo of this can be viewed [here](https://jared-hack-projects.s3.us-east-2.amazonaws.com/employee-skill-searcher/index.html)
